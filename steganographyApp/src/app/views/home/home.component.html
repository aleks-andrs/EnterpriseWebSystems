Steganographer

About:

Steganographer is a web application built using MEAN stack technologies. The main purpose of this web application is to provide a secure steganographic functionality, which is implemented by encoding and decoding of encrypted text payload within given image. The encryption of the payload is handled by Advanced Encryption Standard (AES) algorithm. Additionally, registered Users have an option to store their AES Encryption Keys in an online Password Manager.

Third party content:

This application is utilising MongoDB Atlas cloud database for User data storage.
