Steganographer

About:

Steganographer is a web application built on a MEAN stack. The main purpose of this web application is to provide a secure steganographic functionality, which is implemented by encoding and decoding of encrypted text payload within a given image. The encryption of the payload is handled by Advanced Encryption Standard (AES) algorithm. Additionally, registered Users have an option to store their AES Encryption Keys in an online Password Manager.

Third party content:

This application is utilising a MongoDB Atlas cloud database for User data storage and an external image API service called Lorem Picsum for random stock images retrieval. The application is developed with help of Bootstrap and W3.CSS frameworks, text encryption is implemented using CryptoJS library.
